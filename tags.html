Elemento !DOCTYPE html

Define que el documento se está trabajando en html5

Elemento html

Indica al navegador que todo lo que está abajo es html

Elemento Head

Es donde se guarda la metadata del documento

Elemento meta

Define los metadatos que no pueden ser definidos usando otro elemento html

Elemento title

Indica cual es el título de la página

Elemento body

Representa el contenido principal de documento html

Elemento section

Define una sección en el documento

Elemento nav

Contiene enlaces de navegación

Elemento article

Define contenido que puede existir independientemente del resto

Elemento h1-6

Elementos de cabecera y dependiendo del 1 al 6 va a ser más grande o mas pequeño 1 siendo el más grande

Elemento header

Es la sección de la cabecera de la página

Elemento footer

Es el pie de la página

Elemento p

Indica que se tiene que mostrar un párrafo
